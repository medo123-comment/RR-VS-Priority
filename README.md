# == OS Scheduling Simulator

# 

# This is a Java project that simulates and compares two CPU scheduling algorithms: \*\*Priority Scheduling\*\* and \*\*Round Robin (RR)\*\*.





# == Requirements

# To run this project, you need:

# \-Java Development Kit (JDK): Version 17 or higher.

# \-IDE: IntelliJ IDEA, NetBeans, or Eclipse.

# Build Tool: Maven (integrated in most modern IDEs).

# Library: Standard Java Swing and AWT (for the GUI).







# == Test Scenarios

# We designed these scenarios to test how the algorithms handle different cases:

# \-Scenario A (Basic):\*\* Simple mix of processes to check if the logic is working fine.

# \-Scenario B (Preemption):\*\* A high-priority process arrives later to see if it correctly interrupts the running one.

# \-Scenario C (Fairness):\*\* Testing Round Robin with a fixed Time Quantum to ensure all processes get equal CPU time.

# \-Scenario D (Starvation):\*\* Checking how low-priority processes are affected when many high-priority ones keep arriving.

# 

# &#x20;Error Handling

# The program is protected against common input errors:

# \- E1: Negative priority values are not allowed.

# \- E2: Invalid or zero Burst Times are handled with error messages to prevent crashes.

# 

# \- How to Run

# 1\. Open the folder in \*\*IntelliJ\*\* or \*\*NetBeans\*\*.

# 2\. Make sure it's loaded as a \*\*Maven project\*\* (to get all dependencies).

# 3\. Go to `src/main/java/com/mycompany/os/` and run \*\*Main.java\*\*.
